Calculator API Documentation

 

This API is for a basic calculator. The game simply implements Tic Tac Toe for two players. When you include Calc.js a global variable is created called calculator. All methods should be called on this object. For example, we can insert the number 3 using the calculator.addNumber(3) Below you can find the API:

Method Name Parameters Return Value Description
addNumber(#) #: specifies which button is pressed and should be used in the formula. It should be specified as a number 1-9. The number entered in the parentheses This method allows the user to insert a number in the display, based on the button associated with that number. 
add() The selector is used to select and manipulate HTML element(s) by their Id's. + This method adds the numeric operand to or concatenates with the variable formula. 
subtract() The selector is used to select and manipulate HTML element(s) by their Id's. - This method subtracts the numeric operand or concatenates with the variable formula. 
multiply()  The selector is used to select and manipulate HTML element(s) by their Id's. * This method multiplies the numeric operand or concatenates with the variable formula. 
divide()  The selector is used to select and manipulate HTML element(s) by their Id's. / This method divides the numeric operand or concatenates with the variable formula. 
percent()  The selector is used to select and manipulate HTML element(s) by their Id's. % This method concatenates the modulus so that the remainder can be returned when the first operand is divided by the second operand.
clearBtn()  The selector is used to select and manipulate HTML element(s) by their Id's. None  This method allows the user clear the display UI of the calculator. 
solution()  The selector is used to select and manipulate HTML element(s) by their Id's. The result of the expression This method allows the user to return the solution of "formula" using the eval(formula) function. Eavl will try to convert the string input to a number first, where applicable. The answer is then inserted in the calculator display UI. 
listen(cb) cb: is the callback function. This function will be called whenever a button is pressed. The buttons clicked are passed to the callback function None The listen(cb) functionality is used to track the change in the model and receive notifications. This allows the UI to be updated when the model changes.
notify(num, formula) num and formula are the callback functions. We iterate through the array and call the listen  None Notify all listeners of a change in num and formula
